IDiscountItemIsSelectivelyApplied Property

Whether this discount item is applied selectively to some order items. If true, use TryGetSelectiveDiscountItemSettings(IOrder, IDiscountItem) to get or ChangeSelectiveDiscount(IOrderStub, IDiscountType, IReadOnlyListIOrderProductItemStub, IReadOnlyListIOrderModifierItemStub, IReadOnlyListIOrderCompoundItemComponentStub) to change list of selected order items.
